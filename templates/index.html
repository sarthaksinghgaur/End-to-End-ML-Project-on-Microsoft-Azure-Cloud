Fraud Detection Web App

Powered by Machine Learning and Flask

About This Project

This web application predicts whether a transaction is fraudulent based on transactional features. The model is trained on real-world transaction data using advanced machine learning classifiers for accurate results.

  • Tech Stack: Python, Flask, HTML/CSS, scikit-learn, imbalanced-learn
  • Features: Feature engineering, class balancing, model pipeline, user-friendly interface
  • Use Case: Educational, Fraud Risk Demo, Applied ML Showcase
